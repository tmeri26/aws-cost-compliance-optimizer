# AWS Cost & Compliance Optimizer

An automated cloud governance solution that helps organizations identify cost optimization opportunities, enforce security best practices, and validate Infrastructure-as-Code deployments before they reach production.

## Tech Stack

* AWS
* Terraform
* Python
* GitHub Actions
* Checkov
* Infrastructure as Code (IaC)
* FinOps
* Cloud Governance

## Key Features

* Automated Terraform validation
* Policy-as-Code security scanning
* AWS cost optimization reporting
* Compliance auditing
* CI/CD integration with GitHub Actions
* Infrastructure governance automation
