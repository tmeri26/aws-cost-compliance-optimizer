# AWS Cost & Compliance Optimizer

An automated cloud governance solution that identifies AWS cost optimization opportunities and compliance violations using Terraform, Python, GitHub Actions, and policy-as-code validation.

## Tech Stack

- AWS
- Terraform
- Python
- GitHub Actions
- Infrastructure as Code (IaC)

## Features

- Terraform validation
- Automated CI/CD workflows
- Cloud governance checks
- Cost optimization reporting
- Compliance automation
